Welcome to My Figure Collection app for Android page.

My Figure Collection is a great site where you can manage your manga related figurines and objects collection. In order to keep this amazing service everywhere with you, even without a data connection, I created the first application linked to My Figure Collection, available on Android. This application code is open source, and you can even help me to maintain it :)

Dependances

This project has some dependances to external librairies:

  • ActionBarSherlock by @JakeWharton
  • aQuery by @androidquery
